Personal Projects

CS: Ludo Strategy Board Game

Screenshot of my Ludo CS project. 2021

Ludo Strategy Board Game is my web based strategy board game implemented with OCAML and Javascript. I have added timing, AI-vs-Human mode, and an interactive tutorial quiz to make the game more attractive.

CS: Virtual Risc-V

Screenshot of my Risc-V CS project. 2021

Virtual Risc-V is my cs project which aims at simulated a Risc-V instruction set with C++. The simulated Risc-V could perform complicated mathematical calculations and even approximate Euler's number.

Game: 7-Days Game

Screenshot of my 7-Days game design. 2021

7-Days is my puzzle game made in Unity Engine, which is about a mental patient’s seven-day self-redemption process. Each day represents a game level. Players will advance the story development by solving puzzles in each level.

Game: Lost In Maze

Screenshot of my Lost In Maze game design. 2021

Lost In Maze is my Virtual Reality game made in Unreal Engine. Players can play the game by wearing VR headset. To win the game, the player is transported to a 5-dimension world and need to light up all the targets in the maze.

Data: COVID-19 Transmission Prediction

Screenshot of my COVID-19 data science project. 2021

COVID-19 Transmission and Prediction is my data science project which aims at predicting the COVID-19 Transmission. Data are both self collected and from Israeli Ministry of Health. The model accuracy has been improved from 84% to 96%.

Data: Analysis of Airplane Delays

Screenshot of my Airplane Delays data science project. 2021

This is my data science project which built models in Python to investigate a dataset of 5,000,000 flight delays and cancellations during 2015 in the US. The model resulted in a prediction of more than 90% accuracy.
